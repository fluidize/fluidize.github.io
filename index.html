  • Genetic Algorithms for Algorithmic Trading

    Using python AST trees to represent genes, algorithms can be randomly generated to process logic between technical indicators. Current implementation uses the following: population generation, bayesian optimization(optuna), and staged pruning to find the best performing algorithm within a generated population.

  • Jupiter-Based Live Trading System

    Through the Jupiter Decentralized Exchange API, this system takes in an algorithm and asynchronously feeds market data to it. Market data is streamed from sources such as binance, and trades are processed through the Jupiter DEX API. Features a live paper trading mode for forward testing.

  • Multiheaded Neural Network for Optimal Asset Allocation

    This model uses two heads in a residual-like architecture, both trained on the same data with separate objectives. The distribution head estimates the forward velocity of the asset according to a student-t distribution. The allocation head uses both the distribution and the input data to estimate the optimal percentage allocation of the asset, according to the given loss function.
